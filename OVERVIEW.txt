🎯 WHAT IT DOES
Create cryptographic proofs of data on Solana and Base blockchains.
Any Apify Actor can now prove their data is authentic and untampered.

💡 KEY FEATURES
├─ Multi-chain (Solana + Base)
├─ SHA-256 cryptographic hashing
├─ Ultra-low cost (~$0.001-0.02)
├─ Sub-second finality (Solana)
├─ Verification API included
└─ No custom smart contracts needed

🏗️ ARCHITECTURE
Modular design with chain-agnostic interface:
├─ Core: Hashing, validation, schemas
├─ Chains: Solana adapter, Base adapter
├─ Wallet: Solana keypair, EVM wallet
└─ Verification: On-chain proof checking

👥 TARGET USERS
1. Actor Developers - Add "verified results" to their Actors
2. Enterprise Data Buyers - Compliance and audit trails
3. AI Agents - Observability and accountability
4. DeFi Protocols - Cheap oracle alternative
5. Legal/Journalists - Cryptographic notarization
